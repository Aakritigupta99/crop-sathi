CropSathi — AI Crop Recommendation System
Live: crop-sathi.vercel.app
GitHub: github.com/Aakritigupta99/crop-sathi

- Built end-to-end ML web app recommending crops 
  based on soil & weather data for Indian farmers
- Trained RandomForest classifier (99.55% accuracy) 
  on 2200-sample Kaggle dataset
- FastAPI (Python) ML server integrated with 
  React + Node.js fullstack app
- 10+ pages: Dashboard, AI Chat, Weather, Market, 
  Expense Tracker, Government Schemes
- Deployed: Vercel + Render

Tech: React, Node.js, Express, FastAPI, Python, 
      scikit-learn, MongoDB, Tailwind CSS
